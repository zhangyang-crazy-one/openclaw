## 四、风险评估指标

### 4.1 夏普比率 (Sharpe Ratio)
```
公式: (年化收益 - 无风险利率) / 年化波动率
目标: > 1.0 (优秀), > 0.5 (良好)
```

### 4.2 索提诺比率 (Sortino Ratio)
```
公式: (年化收益 - 无风险利率) / 下行波动率
特点: 只考虑下行风险，更适用于A股
```

### 4.3 Calmar比率
```
公式: 年化收益 / 最大回撤
目标: > 2.0 (保守), > 1.0 (平衡)
```
