# 股票研究报告V3.0生成技能

## 格式要求 (必须严格遵守)

### 8个章节结构 (V3新增利润来源分析)

1. 一、公司概况 - 1.1基本信息(table)+1.2主营业务+1.3主要产品/服务+1.4市场地位
2. 二、商业模式分析 - 2.1商业模式概述+2.2行业地位+2.3竞争优势
3. 三、利润来源分析 (V3新增) - 3.1主营业务利润+3.2投资收益分析+3.3长期股权投资+3.4公允价值变动+3.5其他收益
4. 四、技术面分析 - 4.1技术指标(table)+4.2技术面得分(table)
5. 五、基本面分析 - 5.1财务指标+5.2 Carlson质量评分+5.3巴菲特10大公式+5.4历史分红数据+5.5基本面得分
6. 六、DCF估值模型 - 6.1估值假设+6.2估值结果+6.3 DCF得分
7. 七、行业对比
8. 八、结论 - 8.1综合评分(table)+8.2投资建议

## 利润来源分析 - 数据查询方法 (重要!)

### 查询长期股权投资明细的正确方法

**不要使用 akshare** - akshare无法获取详细的长期股权投资明细

**正确方法: 使用浏览器访问东方财富网**

1. 打开浏览器，访问: `https://emweb.securities.eastmoney.com/pc_hsf10/pages/index.html?type=web&code={股票代码}&color=b#/cwfx`

2. 点击 "财务报表" 标签页

3. 查看资产负债表数据，找到以下项目:
   - **长期股权投资**: 联营/合营企业投资
   - **其他权益工具投资**: 指定为OCI的权益工具投资
   - **交易性金融资产**: 理财产品、证券投资等
   - **投资性房地产**: 出租用的房产/土地

4. 查看利润表，获取投资收益构成:
   - **投资收益**: 包括对联营企业和合营企业的投资收益
   - **公允价值变动收益**: 交易性金融资产公允价值变动
   - **营业外收入**: 政府补助、资产处置收益等

### 利润来源分析内容要求

必须包含:

### 风险提示

当出现以下情况时，需要在报告中添加风险提示:

- 投资收益占比 > 10%: 需关注持续性
- 公允价值变动收益占比 > 5%: 注意市场波动风险
- 交易性金融资产占比 > 30%: 注意资产安全性

## 巴菲特指标修正规则 (重要!)

当投资收益占比过高时，需要对巴菲特10大公式进行修正:

### 投资收益占比判定标准

| 投资收益/净利润占比 | 风险等级 | 修正措施              |
| ------------------- | -------- | --------------------- |
| < 10%               | 低风险   | 无需修正              |
| 10% - 30%           | 中风险   | 扣0.5分/项            |
| 30% - 50%           | 高风险   | 扣1分/项              |
| > 50%               | 极高风险 | 扣2分/项,建议下调评级 |

### 需要修正的指标

1. **ROE修正**: 如果投资收益占比>30%，ROE得分扣1分
   - 原因: ROE可能被投资收益虚高

2. **盈利稳定性修正**: 如果投资收益占比>20%，盈利稳定性扣1分
   - 原因: 投资收益波动大，不稳定

3. **自由现金流修正**: 如果交易性金融资产占比>40%，自由现金流扣1分
   - 原因: 大额理财占用现金流，可能影响主业

4. **资本配置(分红)修正**: 如果主营业务利润为负，分红得分归零
   - 原因: 用投资收益分红不可持续

5. **营业利润率修正**: 如果投资收益占比>30%，营业利润率得分扣1分
   - 原因: 营业利润被投资收益稀释

### 核心利润占比计算

```
核心利润 = 净利润 - 投资收益 - 公允价值变动收益 - 营业外收支
核心利润占比 = 核心利润 / 净利润
```

### 评级调整规则

当投资收益占比过高时，投资建议需要额外调整:

- 投资收益占比 > 30%: 建议下调一级 (如从⭐⭐⭐⭐下调为⭐⭐⭐)
- 投资收益占比 > 50%: 建议下调二级
- 核心利润为负: 直接标记为"不推荐"，需说明"依赖投资收益"

## 历史分红数据查询方法 (重要!)

**必须使用akshare获取真实分红数据，禁止使用模板数据!**

```python
import akshare as ak
df = ak.stock_dividend_cninfo(symbol='股票代码')
print(df)
```

**查询示例**:

- 创业板股票: `ak.stock_dividend_cninfo(symbol='300502')`
- 主板股票: `ak.stock_dividend_cninfo(symbol='600519')`

**输出字段**:

- 实施方案公告日期
- 分红类型
- 送股比例
- 转增比例
- 派息比例
- 实施方案分红说明
- 报告时间

**分红数据必须包含最近5年的记录，并计算累计分红金额**

## 必含字段

### 技术面 (6分)

- Williams %R: <-80得3分
- RSI: <30得1分
- MACD: 金叉得1分
- KDJ: K<20得1分
- 布林: 触及得1分

### 基本面 (7分)

- ROE: >20%得2分, >10%得1分
- 净利润: >1亿得1分
- 毛利率: >30%得1分
- 净利率: >10%得1分
- EPS: >0.3得1分

### DCF (5分)

- 上涨空间 >50%: 5分
- > 30%: 4分
- > 10%: 3分
- > -10%: 2分
- > -30%: 1分

### 巴菲特10大公式 (10分制)

1. 现金测试
2. 负债权益比
3. ROE
4. 流动比率
5. 营业利润率
6. 资产周转率
7. 利息保障倍数
8. 盈利稳定性
9. 自由现金流
10. 资本配置(分红)
