# 统合量化投资分析技能

## 概述

本技能整合行为金融学、货币金融学、量化金融学等多学科研究成果，构建综合投资决策系统。

## 理论基础

### 1. 行为金融学 (Behavioral Finance)
- **投资者情绪**: 过度自信、损失厌恶、羊群效应
- **行为偏差**: 锚定效应、可得性启发、处置效应
- **市场异象**: 动量效应、价值溢价、规模效应

### 2. 货币金融学 (Monetary Finance)
- **货币政策**: 利率、流动性、信贷周期
- **宏观因子**: GDP、通胀、汇率、利率期限结构
- **货币周期**: 流动性宽松/紧缩与资产价格关系

### 3. 量化金融学 (Quantitative Finance)
- **因子模型**: Fama-French五因子、CAPM、动量因子
- **风险模型**: VaR、CVaR、因子暴露
- **组合优化**: 均值-方差、风险平价、凯利公式

## 分析框架

### 综合评分 = α×行为因子 + β×宏观因子 + γ×量化因子

```
行为因子 (30%):
  - 投资者情绪指标: 30%
  - 资金流向: 30%
  - 分析师情绪: 20%
  - 舆情分析: 20%

宏观因子 (30%):
  - 货币环境: 25%
  - 经济周期: 25%
  - 行业周期: 25%
  - 政策影响: 25%

量化因子 (40%):
  - 估值因子: 30%
  - 成长因子: 30%
  - 质量因子: 20%
  - 动量因子: 20%
```

## 使用方法

```bash
python3 unified_quant_analysis.py
```

## 输出

- 综合评分 (0-100)
- 买入/卖出/持有建议
- 风险等级评估
- 行业配置建议

## 决策逻辑

### 买入信号
- 综合评分 > 70
- 宏观因子 > 50 (货币宽松)
- 量化因子无重大风险

### 卖出信号
- 综合评分 < 40
- 宏观因子 < 40 (货币紧缩)
- 估值因子过高 (PE > 80%分位)

### 持有信号
- 综合评分 40-70
- 市场震荡期
- 等待明确信号

## 风险提示

- 历史业绩不代表未来表现
- 模型存在过拟合风险
- 需要定期更新参数
- 建议配合基本面分析使用
